# Racing Taxi Navigation

React-приложение с навигацией для сайта гоночного такси.

## Функции

- Навигация между страницами без перезагрузки
- Подсветка активного пункта меню
- Прямая навигация по URL
- Адаптивный дизайн

## Демо
Посмотреть демо: [GitHub Pages](https://marl3ndev.github.io/racing-taxi-navigation/)

## Начало работы

1. Установите зависимости: `npm install`
2. Запустите приложение: `npm run dev`
3. Откройте [http://localhost:5174](http://localhost:5174) в браузере

## Сборка

Запустите `npm run build` для сборки приложения в продакшн.
